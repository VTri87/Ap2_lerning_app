## KI-Assistent

Der KI-Assistent nutzt die [Anthropic Claude API](https://console.anthropic.com) direkt aus dem Browser. Du brauchst einen eigenen API-Key.

1. Kostenlos registrieren: https://console.anthropic.com
2. API-Key erstellen
3. Auf der Seite unter **⚙ API-Key einrichten** eintragen

Der Key wird nur in deinem Browser gespeichert (localStorage), nicht auf einem Server.
